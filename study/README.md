# MathOntoSpeak Study Package

This folder contains the June 15-June 22 study package for comparing notation-only TTS audio with MathOntoSpeak semantic TTS audio.

## Contents

- `stimuli/study_stimuli.csv` - 10 expressions across algebra, calculus, and linear algebra.
- `stimuli/study_stimuli_equations.csv` - pipeline-ready input for MathOntoSpeak audio generation.
- `instruments/mcq_comprehension_test.csv` - 4 MCQ items per expression, 40 total questions.
- `instruments/nasa_tlx_form.md` - NASA-TLX rating form used after each condition.
- `instruments/post_study_interview_guide.md` - 5-minute interview guide.
- `protocol/counterbalance_schedule.csv` - AB/BA participant order schedule.
- `protocol/study_protocol.md` - concise session procedure.
- `analysis/mathontospeak_study_analysis_template.xlsx` - spreadsheet for comprehension scores, NASA-TLX ratings, interview coding, and summary calculations.
- `audio/notation_only/` - notation-only TTS audio condition.
- `audio/mathontospeak_semantic/` - MathOntoSpeak semantic TTS audio condition.

## Condition Labels

- Condition A: notation-only TTS.
- Condition B: MathOntoSpeak semantic TTS.

Use the counterbalance schedule so half of participants hear A then B, and half hear B then A.
