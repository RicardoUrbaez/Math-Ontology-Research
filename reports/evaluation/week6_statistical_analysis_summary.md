# Week 6 Statistical Analysis Summary

## Comprehension: Paired t-test

- N paired participants: 10
- Notation-only mean accuracy: 35.0% (SD 9.5%)
- MathOntoSpeak semantic mean accuracy: 43.3% (SD 13.5%)
- Mean paired difference: 8.3%
- Paired t-test: t = 1.627, p = 0.138
- Cohen's dz: 0.514

## NASA-TLX: Wilcoxon signed-rank test

- N paired participants: 10
- Notation-only mean workload: 61.7 (SD 8.2)
- MathOntoSpeak semantic mean workload: 64.2 (SD 9.9)
- Mean paired difference: 2.6; negative values favor semantic TTS.
- Wilcoxon signed-rank: W = 19.000, p = 0.734
- Cohen's dz: 0.182

## Thematic Analysis

1. **Semantic role clarity**: Participants described semantic rendering as helpful for identifying the role of mathematical symbols. Code mentions: 15.
2. **Notation-only cognitive load**: Participants described notation-only audio as requiring more memory work or guessing. Code mentions: 13.
3. **Pacing and domain familiarity**: Participants identified pacing and familiarity as factors that affect preferred rendering detail. Code mentions: 2.
